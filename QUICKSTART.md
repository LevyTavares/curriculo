# ⚡ Quick Start - Começar em 5 Minutos

## 1️⃣ Instale as dependências

```bash
npm install
```

## 2️⃣ Inicie o servidor de desenvolvimento

```bash
npm run dev
```

Acesse: `http://localhost:3000`

## 3️⃣ Edite os arquivos

### Seu nome e descrição

📄 `src/components/Hero.astro` (linhas 13-17)

### Seus projetos

📄 `src/components/Projects.astro` (linhas 5-22)

### Suas habilidades

📄 `src/components/Skills.astro` (linhas 5-22)

### Seus dados de contato

📄 `src/components/Contact.astro` (linhas 55, 60, 85)

### Cores e estilo

🎨 `tailwind.config.mjs`

## 4️⃣ Build e Deploy

```bash
npm run build       # Cria versão para produção
npm run preview     # Testa a build localmente
```
