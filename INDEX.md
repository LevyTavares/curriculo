## 🚀 Roteiro Recomendado

### Dia 1: Setup

1. Ler [QUICKSTART.md](QUICKSTART.md)
2. Rodar `npm install`
3. Rodar `npm run dev`
4. Visitar `http://localhost:3000`

### Dia 2: Personalizar

1. Ler [PERSONALIZACAO.md](PERSONALIZACAO.md)
2. Editar informações pessoais
3. Adicionar seus projetos
4. Customizar cores

### Dia 3: Deploy

1. Ler [DEPLOY.md](DEPLOY.md)
2. Escolher plataforma (Vercel recomendado)
3. Fazer deploy
4. Celebrar! 🎉

## 💡 Dicas Rápidas

### Editar conteúdo

```bash
# Seu nome/descrição → src/components/Hero.astro
# Seus projetos → src/components/Projects.astro
# Suas habilidades → src/components/Skills.astro
```

### Ver mudanças

```bash
npm run dev
# Acesse http://localhost:3000
# Mudanças recarregam automaticamente!
```

### Build para produção

```bash
npm run build
npm run preview
```
